What SnapCase Is

Evidence capture with investigative context.

SnapCase helps investigators preserve public social media evidence while they work: profiles, posts, comments, screenshots, media, source links, local files, hashes, bookmarks, and case-ready reports stay organized in one controlled case folder.

SnapCase sits between web evidence preservation tools and enterprise OSINT suites.

It is more intelligence-driven than simple page capture, more evidence-focused than broad source aggregation, and designed around the investigator's live review workflow.

Core Features

Built around the case, not the browser tab.

Every capture is designed to keep source context, local preservation, and review value together.

Profile and post capture

Preserve public accounts, post pages, screenshots, metadata, media files, publication details, and source URLs in a structured case folder.

Comments and replies

Collect relevant comment threads and replies where platforms expose them, keeping authors, timestamps, links, and local report context together.

Bookmarks and selected findings

Mark posts, media, comments, and time moments as case findings, then open platform-specific or global bookmark reports.

Report-ready evidence

Generate investigator-facing HTML reports with case metadata, logos, sections, source links, activity logs, hashes, and captured files.

Media intelligence

Investigator tiers add OCR, keyword checks, video frame extraction, face/object/detail matching, and visual indexing when the case goal calls for it.

Case targets and alerts

Track case keywords, names, identifiers, faces, logos, and recurring media so important matches are surfaced during review.

Investigator Workflow

From public page to organized evidence package.

SnapCase is made for the moment an investigator is already looking at public evidence and needs to preserve it clearly without losing the trail.

01 Open a case

Set the case, investigator, agency, logo, capture goal, and processing level.

02 Capture what matters

Save profiles, posts, videos, comments, screenshots, bookmarks, and selected media.

03 Analyze where needed

Run OCR, keyword review, media matching, frame extraction, and local indexing based on the selected tier.

04 Report the work

Open platform reports, global bookmarks, activity logs, and case summaries with clear source attribution.
